
# Store API

ASP.NET Core Web API for an E-commerce backend.

## Features

- ASP.NET Core Web API
- Entity Framework Core
- SQLite
- ASP.NET Identity
- JWT Authentication
- Role-based Authorization
- Product Management
- Category Management
- Cart System
- Order Checkout
- FluentValidation
- Global Error Handling
- Serilog Logging
- Rate Limiting
- Memory Cache
- Health Checks

## Tech Stack

- .NET 9
- ASP.NET Core
- Entity Framework Core
- SQLite
- JWT
- FluentValidation
- Serilog

## Run Project

```bash
dotnet restore
dotnet run
```
